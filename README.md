ИИ Агент "Феникс". 
Работает на основе Llama 3/Mistral. 
В процессе разработки, работает на пайтоне и парах библиотек, имет доступ к интернету. 
Основная нагрузка идет на железо, из-за выбраной модели.

==========================ПРАВИЛЬНЫЙ ЗАПУСК==================================

1. Первоначально - скачивание библиотек, нужных для запуска бота([LibraryInstall.bat](LibraryInstall.bat))
2. Запуск самого агента([start.bat](start.bat) либо же [startfast.bat](startfast.bat))

============================================================================

Пока запуск только для виндоус.
Написано Дедом(DyDxS1k/The Disgraced One).

===============================ОБНОВЛЕНИЯ===================================

Версия 0.2
1. Добавлена память у ИИ-Асистента.
2. Добавлен Multi-step reasoning.(ИИ Асистент думает в несколько шагов).
3. Добавлени инструменты для анализа веб-ресурсов и файловойй системы.
4. Добавлена паралельная прогрузка инструментов, что ускоряет прогрузку ИИ-Асистента и скорость ответов.
5. Вместо стандартного вывода ответа, ИИ-Асистент вывод ответ в формате JSON, что еще более ускоряет его прогрузку и скорость ответа.

Версия 0.1
1. Переделанная логика ИИ-Агента. (Быстрее отвечает, из-за отсутствия ненужных инструментов для задачи)
2. Убрана уязвимость, при которой использования калькулятора, мог убить систему.
3. Убран хрупкий парсинг.
4. Настроений четкий формат ответа от ИИ-Агента.
